# Analytics Dashboard — Admin (product register)

**Register:** product

## Users

Operators and developers managing the Analytics Dashboard: catalog, sales, access control, and local dev tooling (tunnels, playground).

## Product purpose

Filament admin at `/admin` for back-office tasks on the same Laravel app that powers the public analytics dashboard and shop. Design should feel familiar (Linear/Stripe admin density), not marketing-flashy.

## Brand personality

Calm, capable, operational. Amber primary signals action without shouting. Dark mode supported for long sessions.

## Tone

Direct labels. No filler intros on list pages. Empty states should say what to do next.

## Anti-references

- Hero metrics with gradient accents and decorative motion
- Side-stripe card accents
- Display fonts in tables or form labels
- Reinvented nav patterns

## Strategic principles

- Restrained color: amber for primary actions and selection only
- Predictable Filament layout across all resource groups (Catalog, Sales, Development, Access)
- Local-only dev affordances (tunnel admin, playground) stay in Development group
