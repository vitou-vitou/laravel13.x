Run `/super-spec` with variant `upgrade`.

1. Read `.cursor/skills/spec-kit-openspec-superpowers/references/upgrade-protocol.md`
2. Execute the upgrade protocol:
   - Phase 1: Reconnaissance — latest versions for Spec-Kit, OpenSpec, Superpowers, planning-with-files, ui-ux-pro-max, MemPalace
   - Phase 2: Impact analysis → write findings to `findings.md`
   - Phase 3: Present summary; **user must confirm** before changes
   - Phase 4: Implement approved upgrades only
3. Do not start unrelated feature work during upgrade

Integrated projects registry and severity classification are defined in the upgrade protocol reference.
