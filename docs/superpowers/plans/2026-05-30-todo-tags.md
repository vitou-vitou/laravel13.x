# Todo Tags Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a `tags` JSON array field to todos — store, retrieve, filter, and update tags via the existing REST API.

**Architecture:** Tags stored as a JSON column on the `todos` table, cast to array in the model. No separate tags table. Filter via `?tag=work` query param in the index endpoint.

**Tech Stack:** Laravel 13, SQLite (testing), PHPUnit, `php artisan test`
