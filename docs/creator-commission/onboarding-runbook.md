# Onboarding Runbook — Week 0

**Goal:** Creator is contractually covered, access works, Sheet is live, first batch scope is clear.

**Prerequisites:** Signed distribution agreement (after counsel review), confirmed tier (Lite or Standard).

---

## Checklist

| # | Step | Owner | Done |
|---|------|-------|------|
| 1 | **Kickoff call** — confirm TikTok `@handle`, destinations (YT Shorts + IG Reels), tier, timezone | Operator + Creator | [ ] |
| 2 | **Agreement signed** — store PDF in `{creator_handle}-legal/` (private) | Both | [ ] |
| 3 | **Music policy** recorded in Sheet tab **Onboarding** (skip / replace / creator export) | Operator | [ ] |
| 4 | **YouTube manager access** granted — record Google account email used | Creator | [ ] |
| 5 | **Meta Business Suite** access granted for Instagram — record Meta account email | Creator | [ ] |
| 6 | **Google Sheet created** from [templates/](templates/) — name `{creator_handle}-ops-2026` | Operator | [ ] |
| 7 | **First batch scope** agreed — e.g. last 14 public videos OR new-only from effective date | Both | [ ] |
| 8 | **Asset delivery** — shared Drive folder OR watermark-free export process confirmed | Creator | [ ] |
| 9 | **Weekly batch day** scheduled (e.g. every Tuesday operator builds queue) | Both | [ ] |
| 10 | **Recap email sent** — Sheet link (comment access), approval SLA 24–48h, contact for urgent takedowns | Operator | [ ] |

---

## Kickoff call agenda (~30 min)

1. Confirm handles and platforms (Schedule A)
2. Walk through [pitch-one-pager.md](pitch-one-pager.md) — pricing and commission scope
3. Music policy choice and watermark expectations
4. Approval workflow (Lite = approve all; Standard = spot-check after week 2)
5. First batch size target (pilot week 1: **3–5 videos**, conservative)
6. Q&A

---

## Sheet setup (operator)

1. Create Google Sheet `{creator_handle}-ops-2026`
2. Import CSVs as tabs: **Publish log**, **Weekly metrics**, **Monthly settlement**
3. Add tab **Onboarding** with:

```
creator_handle:
tiktok_url:
youtube_channel_url:
instagram_url:
tier: Lite | Standard
music_policy: skip | replace | creator_export
asset_delivery: Drive link | Telegram | export-only
first_batch_scope:
kickoff_date:
manager_emails_yt:
manager_emails_meta:
```

4. Share with Creator: **Commenter** (approval via comments) or **Viewer** + approve via email/Signal — pick one and document in recap email

---

## Do not

- Share or request **account passwords**
- Publish to YouTube using **watermarked TikTok rips** as master without Creator written OK
- Publish before approval on **Lite** tier
- Promise specific view counts or revenue
- Include TikTok Creator Fund in commission base without Schedule C
